#  Contact Book - Full Stack MERN Application

A full stack Contact Management Application built using **React, Node.js, Express, and MongoDB**.

This project allows users to:

-  Register an account
-  Login securely
-  Add contacts (Name, Phone, Email)
-  View saved contacts in a styled dashboard
-  Logout functionality

This project demonstrates full stack development concepts including authentication, REST APIs, database integration, Password Hashing using bcrypt, and frontend state management.

---

## Features

- User Authentication (Signup / Login)
- Secure Password Handling (bcrypt)
- Add & View Contacts
- RESTful API integration
- Clean project structure
